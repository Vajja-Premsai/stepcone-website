Round 1:
LOGIC MANIA
Three Problem Statements will be given to participants, these problem statements only contain Inputs and outputs rather than the description of the problem. Participants must write a code that satisfies all the test cases for the given input and output.
Round 2:
CODE AUCTION
Code Auction is a team-based event, where each team gets credits of 500 CSI Coins. These coins can be used to buy Problem Statements during the auction of a problem statement. Each problem statement has some base points based on the difficulty of the statement. After a team buys a problem statement, if the team solves the code, then the team’s points will be increased by the base price of the statement. A team cannot buy another code if they didn’t solve at least 40% of the code. After all the codes are sold, the team with the highest points is declared the winner. problem statement to the teams who have registered. They have to come up with a solution and have to build an android app. And after the completion of the time given to them for building their app their team leader have to explain their approach and also have to explain how efficient will their solution will work. They will be assessed by the judges and then the team who scores the highest will be declared as winners.
