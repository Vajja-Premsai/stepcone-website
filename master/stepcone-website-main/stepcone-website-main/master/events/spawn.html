SPAWN THE MACHINE

The candidate must spawn or capture the flag by answering the questions given by HACKTHEBOX. Hack the Box is a website containing machines to be spawned categorized based on topics. Candidate can select any of the machine and continue to answer the questions to reach the practical part. Based on the topic chosen, the candidate has to configure the kali system to obtain a file called root.txt (based on the topic). Candidate have to copy the key specified to capture the flag and that was it. We can have up to 3 rounds, where 2 rounds will be of 45 minutes each and the other one with 60 minutes. In each round, a candidate can spawn machines as much as he/she can within the time limit. The person with more spawns within less time will be qualified for the next round. The last round will be consisting of 2 machines to be spawned specified by the evaluator.
